我更关心可以进入真实工作流的 AI 工具，而不是一次性的 demo。

- **创意 AI 工具**：图像生成、视觉工作流、桌面创作应用。
- **本地 AI 工作站**：语音合成、语音转文字、写作和素材处理工具。
- **Agent 技能资产**：把踩过的坑沉淀成 Codex、CLI 和 agent 可以复用的 skill。
